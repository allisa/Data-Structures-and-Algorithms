# Sorting Algorithms

## Quick Sort

## Challenge
Given an unsorted array, sort the array using quick sort.

## Approach and Efficiency
Big O:
Time: O(n^2) - every item in the array is iterated over
Space: O(1) - the items are trading places in the sort
